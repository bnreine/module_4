1. How do you access the DevTools for Node in Chrome?

type "about:inspect" in google chrome and then click "Open dedicated DevTools
for Node" link.

2. What are the main differences between the regular Chrome DevTools
and the DevTools for Node?

There are no DOM elements or DOM breakpoints because node doesn't deal
with the DOM directly.


3. Using the built-in Node debug tool, what command do you use to continue
the execution of your code.

c (or cont)


4. Which statement do you put in your code to set a breakpoint?

debugger
